*Model · Geneva, Switzerland · 1980s*
**Amos Bairoch** · a hand-checked protein sequence file

**Front.** In 1986, a PhD student in Geneva named [Amos Bairoch](https://pmc.ncbi.nlm.nih.gov/articles/PMC12884847/) was meant to be finishing a thesis, not building a database. Machines were printing out the letter-strings of more and more proteins, but a bare string of letters says nothing about what the protein does or whether it is the same one someone else already filed under another name. So Bairoch read the papers himself and, protein by protein, [typed in by hand](https://www.ebi.ac.uk/about/news/announcements/happy-anniversary-swiss-prot/) a short note on each one's job and parts, with links out to the other files that held it. In [July 1986](https://academic.oup.com/nar/article/24/1/21/2360095) he put out the result and called it Swiss-Prot: a file of nearly [4,000 proteins](https://www.ebi.ac.uk/about/news/announcements/happy-anniversary-swiss-prot/) where a person had checked every line.

**Back.** The file kept growing, and it did not stay his alone. From [1988](https://academic.oup.com/nar/article/24/1/21/2360095) the big sequence lab in Heidelberg joined in, the file later grew into UniProt, and the [rules Bairoch wrote](https://pmc.ncbi.nlm.nih.gov/articles/PMC12884847/) in the 1980s for how to check an entry still run it today. Decades on, DeepMind's AlphaFold learned to guess a protein's folded shape from its letters, and it read those letters straight out of the clean file Bairoch had started. In [2022](https://www.ebi.ac.uk/jdispatcher/blog/2022-07-28-new-afdb-release) it covered over 200 million of them, nearly every protein known to science.
