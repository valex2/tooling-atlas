*Model · Kariya, Japan · 1990s*
**Masahiro Hara (Denso Wave)** · square barcode read from any angle

**Front.** In 1994, an engineer named [Masahiro Hara](https://en.wikipedia.org/wiki/Masahiro_Hara) at [Denso Wave](https://www.qrcode.com/en/history/) in Kariya watched workers on Toyota's parts lines scan box after box by hand. The old stripe barcode held only about [twenty characters](https://www.qrcode.com/en/history/), so one box often wore ten labels, and the beam had to line up straight to read even a single one. Hara laid the data out as black and white cells in a square grid that reads both across and down, and stamped [three nested squares](https://en.wikipedia.org/wiki/QR_code) in the corners as guides, giving each corner mark a stripe pattern of [1:1:3:1:1](https://en.wikipedia.org/wiki/QR_code) that he picked because it almost never turns up elsewhere on a printed page, an idea shaped by the black and white stones of the board game [Go](https://en.wikipedia.org/wiki/QR_code). Now a scanner could spot those three corners first, work out the tilt, and read the whole square from any angle in a flash.

**Back.** For years the square stayed on factory floors and shipping crates, useful only to firms with special readers, though it could hold [thousands of characters](https://en.wikipedia.org/wiki/QR_code) where the stripe code held twenty. Denso Wave owned the [patent but chose not to charge for it](https://www.qrcode.com/en/history/), so anyone was free to print one, and once phones came with cameras the code jumped off the loading dock: [China's payment apps](https://en.wikipedia.org/wiki/QR_code) made it a way to pay, and by 2020 it sat on restaurant tables the world over. The clever part had always been the three corners that let a machine find the code fast; what made it everywhere was the choice to hand the reading rule to everyone.
